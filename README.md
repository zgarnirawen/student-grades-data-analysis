# Student Grades Data Analysis

## Project Description

This project performs a statistical analysis of student grades using Python.

The objective is to extract meaningful insights about academic performance through descriptive statistics, correlation analysis, and visualization.

## Technologies Used

* Python
* pandas
* numpy
* matplotlib
* seaborn
* scipy

## Dataset

The dataset is stored in an Excel file and contains grades of engineering students across multiple subjects.

## Features

### Data Exploration

* Dataset structure and dimensions
* Identification of numerical variables (subjects)
* Number of students and subjects

### Statistical Analysis

* Mean and standard deviation per subject
* Minimum and maximum values
* Interpretation of dispersion

### Outlier Detection

* Detection of abnormal grades using mean ± 2 * standard deviation

### Correlation Analysis

* Correlation matrix between subjects
* Heatmap visualization
* Identification of strongly correlated subjects

### Distribution Analysis

* Histogram of selected subject
* Comparison with normal distribution

### Group Analysis

* Analysis of student performance by group
* Comparison between group average and overall average
